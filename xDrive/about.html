xDrive

xDrive provides access to files in a Blackboard Xythos1 server via the xService RESTful services. Supported files can be downloaded and viewed in app. After a file is downloaded it is cached and can be viewed when offline.

xDrive is designed to be customizable by allowing the organization's Xythos application admin to specify a set of default paths to be created as tab bar items. For instance at ACU, our default paths are Home, Courses, Departments, and Browse. Tab bar items can then be rearranged and saved according to user needs.

Future features:

  • UI style
  • Universal app (iPad support)
  • Modify and delete files/directories
  • Comment and view metadata of files/directories
  • Upload files from Photos
  • Import/export files from Dropbox

xDrive was created by Chris Gibbs as a "10%" side project while working at Abilene Christian University.

xService

xService is an open-source Java webapp that provides the RESTful services for xDrive to access Blackboard Xythos1.

It can be installed on any Xythos application server and customized to the organization's needs. xService was originally developed for ACU's myMobile Files web interface by Harvey McQueen and later customized by Chris Gibbs for use with xDrive.
