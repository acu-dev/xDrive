xDrive

xDrive was created by Chris Gibbs as a "10%" side project while working at Abilene Christian University. It provides access to files in a Blackboard Xythos™ server via the xService RESTful services. Supported files can be downloaded and viewed; cached files can be viewed when offline. xDrive is designed to be customizable by allowing the organization's Xythos application admin to specify a set of default paths to be created as tab bar items. Tab bar items can then be rearranged and saved according to user needs. Future versions will include the ability to upload and delete files as well as comment and view metadata of files.

xService

xService is an open-source Java webapp that provides the RESTful services for xDrive to access Blackboard Xythos™. It can be installed on any Xythos application server and customized to the organization's needs. xService was originally developed for ACU's myMobile Files web interface by Harvey McQueen and later customized by Chris Gibbs for use with xDrive.
